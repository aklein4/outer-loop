# Conditional Latent Variable Notes

Dataset: `aklein4/few-shot-long-test`
Config: `HuggingFaceTB--smoltalk--train`
Split: `train`
Sampling: 10 clusters sampled from cluster IDs `0..99` with `random.Random(42)`.
Generation script: `data_preparation/few_shot/view_clusters.py`

The sampled clusters are `3`, `13`, `14`, `17`, `28`, `31`, `35`, `81`, `86`, and `94`. Each saved JSON contains 64 examples and has only `system`, `user`, and `response` fields. `system` is `null` when no system prompt is present.

This pass looks for latent variables that govern `p(y | x)`: response behavior, transformation rules, output policy, reasoning trace style, tone, format, or other information that changes the distribution over valid responses after the input is known. Pure facts about `p(x)`, `p(y)`, or broad `p(x, y)` subject matter are not counted as defining variables unless they also predict how the answer is written conditional on the prompt.

Theme-fit counts are judgments against the conditional latent variable, counting examples where seeing other examples from the cluster would plausibly help predict `y` given `x`.

## Meta-Learning Assessment

Clusters `013`, `031`, and `014` are the best aligned with the meta-learning objective. Cluster `013` has the most uniform answer template, though much of the signal is also visible from the algebra prompts. Cluster `031` is a particularly strong conditional transformation task: the input already contains the correspondence content, while the hidden cluster variable specifies that the answer should be a concise third-person summary. Cluster `014` is also strong because the latent variable governs explicit surface-form compliance.

Clusters `003`, `028`, `086`, and `094` have moderate conditional signal. Cluster `003` reliably selects interpretive statistics prose, while `028` and `094` often select email summarization. Cluster `086` reveals a planning/decomposition response policy for open-ended goals. These clusters should help a model infer response shape, but some of the behavior can be inferred from the prompt itself.

Clusters `017`, `035`, and `081` are weaker for meta-learning. They contain real response tendencies, but the hidden variable is broad and competes with input-topic regularities. Cluster `035` is a programming-help cluster only for part of its examples, and `081` mixes creative-ideation facilitation with unrelated explanation, role-play, and advice. For stronger few-shot tasks, these clusters would benefit from tighter conditional transformations or hard negatives with the same prompt domains but different output policies.
